# paStormTerm

## setup the repository

`
cd ~

mkdir commands

cd commands

git clone https://github.com/AnthonyFusco/paStormTerm.git
`

## add the commands to your path

add in your shell configuration file (.bashrc or .zshrc)

`
path+=('/home/afusco/commands')

export PATH
`
